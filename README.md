# Dhrumil Patel Portfolio

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Important files

- `app/page.tsx` - main website code
- `app/layout.tsx` - metadata and layout
- `app/globals.css` - Tailwind CSS imports
- `public/profile.jpg` - profile image
- `public/resume.pdf` - replace with real resume
